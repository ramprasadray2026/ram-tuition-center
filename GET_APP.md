# 🚀 RAM TUITION CENTER - ANDROID APP

## 📥 DOWNLOAD OPTIONS

### **OPTION 1: Build APK Yourself (BEST)**
This is the fastest way to get a working APK:

```bash
# Clone repository
git clone https://github.com/ramprasadray2026/ram-tuition-center.git
cd ram-tuition-center

# Install dependencies
npm install -g pnpm
pnpm install

# Build APK (automatic signing included)
chmod +x build-release.sh
./build-release.sh

# APK ready at:
# android/app/build/outputs/apk/release/app-release.apk
```

---

### **OPTION 2: Use Android Studio (EASIEST GUI)**

1. Open Android Studio
2. File → Open → Select `ram-tuition-center/android` folder
3. Wait for Gradle sync
4. Build → Build Bundle(s) / APK(s) → Build APK(s)
5. APK in: `android/app/build/outputs/apk/release/`

---

### **OPTION 3: Manual Gradle Build**

```bash
cd android
./gradlew assembleRelease

# APK at: app/build/outputs/apk/release/app-release.apk
```

## 📱 INSTALL APK

Once you have `app-release.apk`:

1. Transfer to Android phone
2. Settings → Security → Enable "Unknown Sources"
3. Tap APK file
4. Install
5. Open "RAM Tuition" app
