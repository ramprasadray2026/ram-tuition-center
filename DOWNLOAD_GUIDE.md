## 🚀 Build from Source (Optional)

If you want to build the APK yourself:

```bash
# 1. Clone repository
git clone https://github.com/ramprasadray2026/ram-tuition-center.git
cd ram-tuition-center

# 2. Install dependencies
npm install -g pnpm
pnpm install

# 3. Build APK
chmod +x build-release.sh
./build-release.sh

# 4. APK will be at:
# android/app/build/outputs/apk/release/app-release.apk
```

## 🆘 Troubleshooting

### "App not installed"
- Enable "Unknown Sources" in Settings
- Ensure file downloaded completely
- Check Android version (5.1+)

### "Installation blocked"
- Go to Settings → Security
- Enable installation from unknown sources
- Try again

### "App won't start"
- Restart your phone
- Clear app cache: Settings → Apps → RAM Tuition → Clear Cache
- Reinstall the app

### "Upload fails"
- Check internet connection
- File size must be < 50MB
- Supported formats: PDF, DOC, DOCX, JPG, PNG
