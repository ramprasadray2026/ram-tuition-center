## 📥 How to Get the APK

### Option 1: Direct Download from Repository
Go to: https://github.com/ramprasadray2026/ram-tuition-center

**Files to Download:**
1. Clone or download the repository as ZIP
2. Extract the files
3. Navigate to: `android/app/build/outputs/apk/release/`
4. Find: `app-release.apk`

### Option 2: Build APK Yourself (Recommended)

**Prerequisites:**
- Node.js 16+
- Java 11+
- Android SDK

**Build Steps:**
```bash
# Clone repository
git clone https://github.com/ramprasadray2026/ram-tuition-center.git
cd ram-tuition-center

# Install dependencies
npm install -g pnpm
pnpm install

# Build APK automatically
chmod +x build-release.sh
./build-release.sh
```

**APK will be at:** `android/app/build/outputs/apk/release/app-release.apk`

## 📱 Installation Instructions

1. **Download** the `app-release.apk` file
2. **Transfer** to your Android phone
3. **Open Settings** → **Security** → Enable **"Unknown Sources"**
4. **Open** the APK file
5. **Tap "Install"**
6. **Done!** Open the app from your app drawer

## 🆘 Troubleshooting

### "Installation failed"
- Enable Unknown Sources in Settings
- Check Android version (5.1+)
- Free up storage space

### "App won't start"
- Clear app cache
- Reinstall the app
- Restart your phone

### "Upload fails"
- Check file size (< 50MB)
- Check file type (PDF, DOC, DOCX, JPG, PNG)
- Check internet connection
